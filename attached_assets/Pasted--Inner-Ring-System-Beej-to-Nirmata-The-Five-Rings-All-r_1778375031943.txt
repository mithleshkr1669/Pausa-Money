# Inner Ring System — Beej to Nirmata

## The Five Rings

All ring names are in Hindi — this is deliberate. It creates cultural identity no Western platform can copy.

| Ring | Hindi | English | Cap | Entry Method | ring_tier value |
|------|-------|---------|-----|-------------|-----------------|
| 1 | Beej (बीज) | The Seed | Unlimited | Automatic on signup | 1 |
| 2 | Saathi (साथी) | The Companion | Unlimited | Criteria met automatically | 2 |
| 3 | Sevak (सेवक) | The Servant | 201 | Application required | 3 |
| 4 | Sutradhaar (सूत्रधार) | Thread Holder | 51 | Founder invite only | 4 |
| 5 | Nirmata (निर्माता) | The Creator | 11 (FOREVER) | Founder selection | 5 |

**Critical:** The cap of 11 Nirmatas is permanent and sacred. Never allow more. Display the cap publicly.

## Ring Progression Logic

### Beej → Saathi (automatic, tracked by ring_activity)
```typescript
// Check on every activity event
async function checkSaathiEligibility(userId: string) {
  const { data } = await supabase
    .from('ring_activity')
    .select('activity_type, created_at')
    .eq('user_id', userId)

  const upvotesReceived = await supabase
    .from('votes')
    .select('count')
    .eq('target_type', 'answer')
    // join to answers where user_id = userId

  const daysActive = daysSinceFirstActivity(data)
  const helpfulVotes = upvotesReceived.count

  if (helpfulVotes >= 30 && daysActive >= 30) {
    await promoteToSaathi(userId)
  }
}
```

### Saathi → Sevak (application-based)
Application form fields:
- Finance background (optional — CA student, finance grad, experienced investor, other)
- Why do you want to be a Sevak? (200 words)
- Commit to reviewing 5 plans per week (checkbox)
- Pass 10-question India finance quiz (score ≥ 7/10 to pass)

Quiz topics: SIP, EPF vs PPF, Section 80C, term insurance, direct vs regular MF, emergency fund, CIBIL score, tax regime choice, NPS basics, ELSS lock-in.

Review by 3 existing Sevaks (not just founder). Majority vote.

### Sevak → Sutradhaar (invite only)
- Founder identifies exceptional Sevaks after 6+ months
- Specific role assigned (not generic upgrade): "I want you to lead the Bihar chapter" or "I want you to run Hindi content"
- Offer made privately before public announcement
- Signs simple partnership agreement (revenue share terms, conduct expectations)

### Sutradhaar → Nirmata (founder selection)
- 7 slots: earned through community leadership over 12+ months
- 2 slots: domain experts recruited directly (CA, SEBI RIA)
- 2 slots: founder reserves — chosen last, after seeing who community can't function without
- Once a Nirmata, always a Nirmata (irrevocable except for serious violations)

## Founding Wall (on /about page)

Show all 11 Nirmata slots. Empty slots show "To be filled — X remaining." When all 11 are filled, lock and announce.

```tsx
// This is the most important trust signal on your platform
// Every new member sees who built this community
export function FoundingWall({ nirmatas }: { nirmatas: Profile[] }) {
  const slots = Array(11).fill(null).map((_, i) => nirmatas[i] || null)
  return (
    <section>
      <h2>निर्माता · Founding Council</h2>
      <p>11 people. Forever. These are the founders of the FinCoach community.</p>
      <div className="founding-grid">
        {slots.map((n, i) => n
          ? <NirmataCard key={i} profile={n} index={i + 1} />
          : <EmptySlot key={i} index={i + 1} remaining={11 - nirmatas.length} />
        )}
      </div>
    </section>
  )
}
```

## Anti-patterns (Never Do These)

- Never auto-promote anyone to Sevak or above without human judgment
- Never expand the Nirmata cap — not even temporarily
- Never publicly demote anyone — handle privately
- Never give titles to silence someone or manage a conflict
- Never make FinCoins the only reason to contribute — identity must come first
