# Pausa Homepage — Pre-Auth Landing Page

## What This Page Must Do

This is the **only page a stranger sees before deciding to trust you**. It must do five things in order:

1. **Earn attention** in 3 seconds (headline + subheadline)
2. **Create recognition** — "this was made for me" (origin story + vulnerable moment)
3. **Build trust** — real people, real stories, verified experts
4. **Show the system** — the rings, the AI, the community
5. **Convert** — one clear CTA: "Get my free financial plan"

The page ends at the signup/login button. Nothing after that belongs here.

---

## Aesthetic Direction

**Theme:** Warm, handmade, honest — like a letter written by someone who grew up in Bihar and figured things out the hard way. NOT a SaaS product page. NOT a bank. NOT a fintech startup with gradients and rocket emojis.

**Palette:**

```css
:root {
  /* Backgrounds — warm paper, not cold white */
  --bg-primary: #faf7f2; /* warm off-white, like old paper */
  --bg-secondary: #f2ede3; /* slightly deeper warm */
  --bg-dark: #1a1612; /* deep warm dark for contrast sections */
  --bg-card: #ffffff;

  /* Ink — warm blacks */
  --ink-primary: #1c1814;
  --ink-secondary: #4a4438;
  --ink-muted: #8c8070;
  --ink-faint: #c4b9aa;

  /* Accent — saffron India */
  --saffron: #e8651a;
  --saffron-soft: rgba(232, 101, 26, 0.1);
  --saffron-glow: rgba(232, 101, 26, 0.22);

  /* Trust signals */
  --moss: #3d6b4f; /* verified / trusted green */
  --moss-soft: rgba(61, 107, 79, 0.1);
  --gold: #c49a1a; /* Nirmata / founding gold */
  --gold-soft: rgba(196, 154, 26, 0.1);

  /* Border */
  --border: rgba(28, 24, 20, 0.1);
  --border-strong: rgba(28, 24, 20, 0.2);
}
```

**Typography:**

```css
/* Display — Lora (serif, warm, literary) */
@import url("https://fonts.googleapis.com/css2?family=Lora:ital,wght@0,400;0,500;0,700;1,400;1,500&family=Plus+Jakarta+Sans:wght@300;400;500;600&family=JetBrains+Mono:wght@400;500&display=swap");

--font-display: "Lora", Georgia, serif; /* headlines, pull quotes */
--font-body: "Plus Jakarta Sans", sans-serif; /* body text, UI */
--font-mono: "JetBrains Mono", monospace; /* labels, stats, badges */
```

**Motion:** Subtle scroll-triggered fade-ins. No bouncing. No confetti. The page should feel like turning pages of a well-made book — deliberate, unhurried.

## Copy Rules (Enforce in All Code)

| ❌ Never write       | ✅ Write instead                          |
| -------------------- | ----------------------------------------- |
| "financial advice"   | "financial guidance"                      |
| "we advise you"      | "based on your situation, consider"       |
| "best investment"    | "commonly recommended for your situation" |
| "guaranteed"         | (never say this)                          |
| "Sign up" (generic)  | "Get my free plan"                        |
| "Join our community" | "Find people who get it"                  |
| Stock photo faces    | Illustrated avatars or no photos          |
| "AI-powered"         | "Built with AI, reviewed by humans"       |

## Conversion Logic

**Primary CTA everywhere:** "Get my free financial plan →"

- Saffron background (#e8651a), white text, 16px font, 48px height, pill border-radius
- On click: goes to `/plan` (the AI intake form) — NO signup wall before the plan

**Secondary CTA (navigation + footer):** "Sign in"

- Ghost button, saffron border, saffron text

**The most important conversion principle:**

> Let the user see the AI plan BEFORE asking them to sign up.
> The plan is the product demo. Show it first. Ask for account after.
> Auth wall = only when saving the plan. Not before.

## Responsive Breakpoints

```css
/* Mobile first */
/* sm: 640px — cards stack, hero text smaller */
/* md: 768px — 2-column grids appear */
/* lg: 1024px — full 3-column layouts */
/* xl: 1280px — max-width container, breathing room */

.container {
  max-width: 1200px;
  margin: 0 auto;
  padding: 0 20px;
}
```

**Mobile priorities:**

- Hero headline readable at 320px (clamp font size)
- Story cards: horizontal scroll, snap scrolling
- Ring diagram: simplified vertical list (not SVG)
- CTA button: full width on mobile

---

## Accessibility

- All section labels as real `<h2>` or `<h3>` (not just styled divs)
- CTA buttons: `aria-label="Get my free financial plan"`
- Color contrast: all text passes WCAG AA (warm dark on warm light ✓)
- Hindi text: `lang="hi"` on Hindi copy elements
- Animated elements: `prefers-reduced-motion` media query stops all animations
- Alt text on all SVG illustrations

---

## What NOT to Build on This Page

- Login/signup form (goes on `/login`)
- Dashboard preview screenshots
- Pricing table (goes on `/pricing`)
- Blog posts or articles
- App download badges
- Cookie banner (handle separately)
- Live chat widget
- Pop-ups or modals of any kind
- Anything that requires the user to be logged in
